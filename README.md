# Complete DevOps CI/CD Pipeline

This project implements a complete DevOps CI/CD pipeline to automate application build, testing, deployment, and monitoring.

## Tools Used
- GitHub
- Jenkins
- Docker
- Kubernetes
- Prometheus
- Grafana
- AWS/GCP

## Workflow
1. Code pushed to GitHub
2. Jenkins pipeline triggered
3. Build and test executed
4. Docker image created and pushed
5. Kubernetes deployment updated
6. Monitoring via Prometheus
7. Visualization via Grafana

## How to Run
- Configure Jenkins
- Add Docker & Kubernetes credentials
- Run pipeline
